# Sweetalert

An awesome replacement for JavaScript's alert.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'sweetalert', git: 'https://github.com/roshan92/sweetalert.git'
```

And then execute:

    $ bundle
